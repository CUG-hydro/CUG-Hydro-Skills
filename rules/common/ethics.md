# 科研伦理规则

## 数据诚信

- 不伪造、篡改数据
- 不选择性报告结果（cherry-picking）
- 异常值处理必须透明说明
- 保留原始数据和处理脚本

## 引用规范

- 所有引用必须亲自阅读或明确标注为间接引用
- 不夸大或歪曲他人研究结论
- 自引必须合理且必要
- 引用与论文内容相关

## 作者署名

- 所有作者必须对研究有实质性贡献
- 致谢部分明确标注资助来源
- 利益冲突声明

## 代码与数据共享

- 鼓励开源代码（选择合适的License）
- 数据共享遵循FAIR原则
- 保护隐私和敏感信息
- 遵守机构数据管理政策

## AI使用声明

- 明确披露AI工具（如Claude）的使用范围
- AI辅助写作需人工审核准确性
- 不将AI生成内容作为原创研究发现
